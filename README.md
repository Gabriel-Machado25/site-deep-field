# 🌌 Deep Field - Landing Page

Landing Page institucional e de alta conversão desenvolvida sob medida para a agência **Deep Field** (Vendas, Marketing e Inovação). O projeto foi estruturado do zero com o objetivo de criar uma interface imersiva, moderna e totalmente focada em experiência do usuário (UX) e conversão.

## 🎨 Design & Conceito
O projeto foi construído aplicando princípios de psicologia das cores e UI avançada:
* **Identidade Visual:** Paleta imersiva baseada em tons profundos de azul-noturno e preto, com pontos de foco estratégico (acentos) em dourado para guiar a atenção do usuário para os CTAs.
* **Estética Orgânica:** Uso de espaçamentos generosos (*white space*) e efeitos visuais modernos como Glassmorphism para fugir de layouts genéricos e engessados.

## 🛠️ Tecnologias Utilizadas
Para garantir máxima performance, carregamento rápido e sem dependências pesadas, o projeto foi desenvolvido utilizando tecnologias nativas:
* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Layout responsivo (Mobile First), variáveis globais, Flexbox/Grid e animações fluidas de micro-interação.
* **JavaScript (ES6+):** Manipulação de eventos, efeitos de scroll e interações dinâmicas.
